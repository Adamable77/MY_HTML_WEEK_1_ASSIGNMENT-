HTML SIMPLIFIED

Html is a markup language for describing web pages. Html stands for Hyper Text Markup Language, a Markup language is a set of markup tags

HTML tags are keywords surrounded by angle brackets like HTML tags normally come in pairs like and The first tag in a pair is the start tag, the second tag is the end tag Start and end tags are also called opening tags and closing tags.

  • An unordered list can be specified with the tags <ul> and </ul>.
  • An unordered list typically contains a number of list items that can be specified with tags <ul> and </ul>.
  • After you have created your unordered list, check out what happens when you convert it to ordered list by replacing the tags <ul> and </ul> with <0l> and </ol>, respectively
